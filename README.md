# java-stepGame

게임을 승리 할때 마다 단계가 오른다.

#### 1단계 게임 - 가위 바위 보 게임
사용자는 1(가위) 2(바위) 3(보) 를 입력할수 있다.<br>
컴퓨터랑 3세판 을 해서. 1번이라도 이기면 단계가 올라간다. <br>
<br>
<br>

#### 2단계 게임 - UP and Down
컴퓨터는 1 ~ 100 까지의 임의의 숫자를 하나 고른다. <br>
사용자는 1 ~ 100 까지의 숫자를 입력한다.  <br>
컴퓨터의 정답 보다 숫자가 작으면 up ,  컴퓨터의 정답 보다 숫자가  높으면 down 을 출력한다.  <br>
8번(포함) 안에 성공하면 단계가 올라간다. <br>
<br>
<br>
#### 3단계 게임 - 가위 바위 보,로 계단 오르기 게임
사용자는 1(가위) 2(바위) 3(보) 를 입력할수 있다.<br>
승리한 사람이 3계단을 올라간다. <br>
진 사람은 1계단을 내려간다.<br>
20계단을 먼저 올라가면 승리한다. <br>


### 입출력 요구 사항
#### 입력

+ 1 단계 a - 처음에 사용자의 이름을 입력 받는다.(5글자 내외, 영어만 및 숫자만, 처음에 숫자는 불가능)
+ 1 단계 b - 사용자는 숫자 1,2,3 중 하나만 입력한다.
+ 2 단계 - 사용자는 1 ~ 100 사이의 숫자만 입력 가능하다.
+ 3 단계 - 사용자는 숫자 1,2,3 중 하나만 입력한다.

#### 출력
+ 1 단계 a - 사용자 에게 "가위(1),바위(2),보(3)를 내주세요 - (숫자만 가능)" 라는 문구를 출력한다
+ 1 단계 b - 사용자가 숫자 1,2,3 입력후 컴퓨터는 가위, 바위 ,보 하나를 출력한다.
+ 1 단계 c - 출력후 무승부 이거나 사용자가 지면 "졌습니다." 라는 메시지 후 "1단계 a"를 다시 출력한다.
+ 1 단계 d - 출력후 이기면 승리 메시지를 보내고 단계가 올란간다.
+ 1 단계 e - 3세판 모두 졌을 경우, 사용자 이름 및 단계를 보여주고 게임을 종료한다
<br>
+ 2 단계 a - 사용자에게 "1 ~ 100 까지의 숫자를 입력하세요" 라는 문구를 출력한다.
+ 2 단계 b - 사용자의 숫자에 따라 "up" or "down"을 출력한다. 그리고 다시 "2단계 a"를 다시 출력한다.
+ 2 단계 c - 8번(포함) 안에 답을 맞추면 다음 단계로 올라간다.
+ 2 단계 d - 8번에 못 맞출 경우 "숫자를 못 맞추쳤습니다.\n 컴퓨터의 정답 : @@ " 라는 메시지후 몇단계 까지 올라 갔는지 사용자 이름 및 단계를 보여주고 게임을 종료한다
<br>
+ 3 단계 a - 사용자 에게 "가위(1),바위(2),보(3)를 내주세요 - (숫자만 가능)" 라는 문구를 출력한다
+ 3 단계 b - 사용자가 숫자 1,2,3 입력후 컴퓨터는 가위, 바위 ,보 하나를 출력한다.
+ 3 단계 c - 무승부 이면 "무승부 입니다." 라는 메시지 출력 후, 사용자 및 컴퓨터의 현재 상태를 보여준다.
+ 3 단계 d - 사용자나 컴퓨터가 이길 경우 "@@@ 이 이기셨습니다.\n @@@은 3칸 전진" 라는 메시지 출력후, 이긴자는 +3이 추가 되고 진 자는 -1이 추가 된다.(0일 경우 그대로)
+ 3 단계 e - 사용자가 20계단을 먼저 오르면 "모든 게임에 승리 하셧 습니다" 라는 메시지 출력 후 "사용자 이름 및 단계를 보여주고 게임이 종료된다."
+ 3 단계 f - 사용자가 졌을 경우 "아쉽 습니다." 라는 메시지 후 사용자 이름 및 단계를 보여주고 게임을 종료한다
